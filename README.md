# [임시] FireGenerators - 금융 지식 학습 및 투자 코칭 웹 서비스

<img src="my_app\assets\FIRE_LOGO_large.png" width="200"/>  


2030 세대가 자신의 **투자 성향**과 **금융 지식 수준**을 진단하고, 맞춤형 학습·추천을 받을 수 있는 **AI 기반 금융 코칭 플랫폼**입니다.

## 1. 프로젝트 소개

### 1.1. 개발배경 및 필요성

1. **MZ 세대의 투자 관심 증가**

   * 한국금융연구원(2023)에 따르면, 2030 세대의 68%가 주식·ETF 등 금융투자 경험을 보유.
   * 하지만, 기초 지식 부족으로 **단기 투자 실패 경험**이 많음.

2. **금융 교육의 부족**

   * 학교 교육에서는 금융 지식을 배우기 어려움.
   * 온라인 강의나 콘텐츠는 **개인 맞춤형이 부족**하여 흥미와 지속성이 떨어짐.

3. **AI 기반 맞춤형 학습 수요 증가**

   * ChatGPT, Gemini 등 LLM 활용이 일상화됨.
   * 개개인의 성향, 지식 수준에 맞춘 금융 교육 서비스가 필요.

→ **금융 지식 학습 + 맞춤형 투자 성향 분석 + AI 코칭**을 결합한 플랫폼의 필요성이 제기됨.

### 1.2. 개발 목표

**“파이어제너레이터는 사용자의 투자 성향과 금융 지식 수준을 기반으로 맞춤형 학습과 주식 코칭을 제공하는 AI 금융 코치 서비스”**

* 투자 성향 분석 → 맞춤형 금융 퀴즈 및 학습 제공
* 금융 관심사 기반 → 맞춤형 금융 지식 카드 추천
* 투자 수준 기반 → 주식 종목 추천 및 포트폴리오 코칭


## 2. 주요 기능

### 2.1. 오늘의 금융 퀴즈

* 사용자의 **관심 카테고리**와 **지식 수준**에 맞춘 OX·객관식 퀴즈 자동 생성.
* 문제 풀이 → 난이도 자동 조정 → 최종 수준 평가(초급·중급·상급).
* 퀴즈 결과는 LLM이 **3문장 요약 리포트**로 제공.

### 2.2. 맞춤형 금융 지식 카드

* 사용자가 선택한 관심 분야(ETF, 부동산, 연금, FIRE 등)에 맞는 카드 콘텐츠 제공.
* LLM 기반 요약 + 출처 제공으로 학습 신뢰도 강화.

### 2.3. 투자 성향 기반 추천

* 투자 성향 테스트(신중함, 불안, 아쉬움, 욕심, 자신감, 혼란 등 지표).
* 성향 결과에 따라 **안정형·적극형·공격형** 상품 추천.
* 국내 ETF, 주식, 펀드 정보와 연계.

### 2.4. 보유 주식 AI 코칭

* 사용자가 보유 종목 등록 → AI가 현황 분석 및 개선 코칭 제공.
* 종목별 피드백(리스크 요인, 대체 투자 추천 등).


## 3. 시스템 구성

### 아키텍처

* **Frontend**: Streamlit
* **Backend**: FastAPI, Python
* **Database**: Supabase (PostgreSQL + Vector DB)
* **AI/ML**: OpenAI GPT-5, Gemini, KoBERT, Upstage Embeddings
* **Infra**: AWS EC2, GitHub Actions


## 4. 차별성

1. **금융 학습 + 투자 코칭 통합**

   * 기존 금융 퀴즈 앱은 단순 지식 테스트.
   * 기존 투자 앱은 단순 시세 제공.
   * **학습 + 투자 피드백**을 한 번에 제공하는 플랫폼.

2. **사용자 맞춤화**

   * 관심사 기반 퀴즈·지식 추천.
   * 투자 성향 기반 상품 추천.

3. **AI 기반 요약·추천**

   * 퀴즈 결과를 AI가 3문장으로 요약 → 사용자가 자신의 금융 수준을 쉽게 이해.
   * 보유 종목에 대한 AI 코칭 → 기존 HTS·MTS와 차별화.
