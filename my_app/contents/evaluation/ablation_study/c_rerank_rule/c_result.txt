# C. 룰 기반 리랭킹 계수 조정 실험 ⚖️

## 🎯 실험 목적
수치 기반 리랭킹 공식의 최적 가중치(α, β, γ) 조합 탐색

## 📐 리랭킹 공식
```
Score_final = α × Score_vector + β × Score_level + γ × Score_tag
```

## 🧪 실험 설계

### C1~C5: 가중치 조합 비교

#### C1: 벡터 중심 (현재 설정)
- α(벡터): **0.6** 
- β(레벨): **0.3**
- γ(태그): **0.1**

#### C2: 균형형
- α(벡터): **0.5**
- β(레벨): **0.3** 
- γ(태그): **0.2**

#### C3: 레벨 중심
- α(벡터): **0.4**
- β(레벨): **0.5**
- γ(태그): **0.1**

#### C4: 태그 중심  
- α(벡터): **0.4**
- β(레벨): **0.2**
- γ(태그): **0.4**

#### C5: 극단형
- α(벡터): **0.8**
- β(레벨): **0.1**
- γ(태그): **0.1**

### 측정 지표
- **추천 정확도**: 사용자 선호도와의 일치율
- **다양성 점수**: 추천 결과의 다양성
- **개인화 수준**: 사용자별 맞춤 정도
- **계산 속도**: 리랭킹 처리 시간
