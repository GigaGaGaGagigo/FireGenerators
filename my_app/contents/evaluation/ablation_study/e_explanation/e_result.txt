# E. 프롬프트 전략 실험 💬

## 🎯 실험 목적
사용자 맞춤형 콘텐츠 설명 생성을 위한 프롬프트 전략 최적화

## 🧪 실험 설계

### E1~E3: 프롬프트 전략 비교

#### E1: Generic (범용형)
```
당신은 금융 전문가입니다. 
다음 콘텐츠를 일반 사용자가 이해하기 쉽게 설명해주세요.
```
- **장점**: 단순하고 일관된 설명
- **단점**: 개인화 부족, 획일적 접근

#### E2: Level-Specific (정적 레벨별)  
```
당신은 금융 전문가입니다.
다음 콘텐츠를 {레벨} 수준의 사용자가 이해할 수 있도록 설명해주세요.

- Beginner: 기초 용어부터 친근하게
- Intermediate: 실용적 활용법 중심으로  
- Advanced: 심화 분석과 전략적 관점으로
```
- **장점**: 지식 수준별 맞춤 설명
- **단점**: 감정 상태와 관심사 미반영

#### E3: Adaptive (동적 적응형)
```  
당신은 금융 상담 전문가입니다.
사용자 정보:
- 지식 수준: {level}
- 투자 감정: {emotions}점 ({emotion_status})
- 관심 분야: {interest_tags}
- 위험 성향: {risk_tolerance}

위 정보를 바탕으로 다음 콘텐츠를 사용자에게 
가장 적합한 방식으로 설명해주세요.
```
- **장점**: 완전한 개인화, 감정과 상황 고려
- **단점**: 복잡한 프롬프트, 응답 시간 증가

### 측정 지표
- **이해도**: 사용자 콘텐츠 이해 수준
- **만족도**: 설명의 유용성 평가
- **참여도**: 추가 질문 및 상호작용 횟수
- **응답 속도**: 설명 생성 시간

## 💡 프롬프트 최적화 요소

### 개인화 변수
```python
user_context = {
    "level": "Beginner|Intermediate|Advanced",
    "emotions": -50~+50,  # 투자 심리 점수
    "emotion_status": "매우 긍정적|긍정적|중립적|불안|매우 불안",
    "interest_tags": ["은퇴 준비", "주식 투자", "펀드"],
    "risk_tolerance": 0~100,  # 위험 허용도
    "investment_goal": "안정 수익|성장|공격적 투자"
}
```

### 감정별 설명 전략
```
😊 매우 긍정적 (+30 이상):
- 도전적 투자 기회 중심 설명
- 성장 잠재력과 기회 강조

🙂 긍정적 (+10~+29):  
- 균형잡힌 관점으로 설명
- 기회와 위험 동시 언급

😐 중립적 (-10~+9):
- 객관적이고 교육적인 설명
- 기본 개념부터 차근차근

😟 불안감 (-30~-11):
- 안정성과 안전성 강조
- 리스크 관리 방법 중심

😔 매우 불안 (-30 이하):
- 보수적 접근법 제시  
- 단계별 천천히 진행
```

## 📈 예상 실험 결과

### 가설
1. **E3 (적응형)**: 최고 만족도, 느린 속도
2. **E2 (레벨별)**: 균형잡힌 성능  
3. **E1 (범용)**: 빠른 속도, 낮은 개인화

### 트레이드오프
- **개인화 vs 속도**: 맞춤 설명일수록 생성 시간 증가
- **복잡성 vs 일관성**: 복잡한 프롬프트일수록 결과 변동성 증가


## 🚀 구현 계획

### Phase 1: 기본 템플릿
- [ ] E1, E2, E3 프롬프트 템플릿 개발
- [ ] 동적 변수 주입 시스템 구축

### Phase 2: 실험 실행
- [ ] 사용자 그룹별 A/B/C 테스트
- [ ] 만족도 피드백 수집 시스템

### Phase 3: 최적화
- [ ] 최고 성능 전략 선정
- [ ] 프롬프트 템플릿 fine-tuning
- [ ] 실시간 적용 및 모니터링

## 💬 현재 프롬프트 전략
기본적으로 **E2 (레벨별)** 방식을 사용하되, 사용자 감정 상태가 극단적일 경우 **E3 (적응형)** 요소를 부분 적용하는 하이브리드 접근 방식 권장
